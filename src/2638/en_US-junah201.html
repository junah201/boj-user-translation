Cheese

Description

A very thin layer of cheese is placed on an N×M grid as shown in Figure 1. Here, N represents the number of rows, and M represents the number of columns. The cheese must be kept frozen; otherwise, it melts when exposed to room temperature. Each cheese grid cell (a small square) will melt if at least two of its four sides are in contact with the air, and this happens exactly one hour after exposure.

Figure 1

In Figure 2, the space inside the cheese is not in contact with the air, so the cheese cells within this space do not melt. Only the cheese cells marked with 'C' will melt. After one hour, if air reaches the inner space, cheese cells will melt as shown in Figure 3.

Figure 2

Figure 3

The edges of the grid are assumed to have no cheese. Your task is to write a program that calculates the exact time it takes for all the cheese to melt.

Input

The first line contains two integers, N and M (5 ≤ N, M ≤ 100), representing the size of the grid. The following N lines describe the grid, with '1' representing a cell with cheese and '0' representing a cell without cheese. Each value is separated by a space.

Output

Output the exact number of hours it takes for all the cheese to melt.
